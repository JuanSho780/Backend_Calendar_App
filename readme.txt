
# 🌱 Proyecto EcoWare — Sistema de Registro y Gestión de Usuarios

Este proyecto implementa un sistema web que permite el registro, autenticación y gestión de usuarios, combinando un backend robusto en **FastAPI** y un frontend moderno en **React**.  
El objetivo principal es ofrecer una arquitectura limpia, escalable y fácil de mantener, aplicando patrones de diseño como **Singleton**, **Factory**, **Facade** y **Command**.
